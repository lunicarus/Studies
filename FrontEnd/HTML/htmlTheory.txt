docktype: version of HTML <!DOCKTYPE html>

head: metadata (encription,language,data,suporting files)
- <title></title>,
- <meta charset="UTF-8">,
- <link rel="stylesheet" href="">,

body: displayable content 

tags: define the beggining and the end of an element
<main> </main>

element: everything from the start tag to the ending tag
<br>,
<span>"hello"</span>,
<div></div>

block tags: take a definite amount of space (Headings,Paragraphs,Divs)
inline tags: only uses as much space as needed to contain the element

class - applies special properties to groups of elements
id - specifies a single element
style - specifies a certain
acessley - shorcut key to activate an element
tabindex - order of elements by using a TAB index

attributes - provide additional information about elements
src="" - specifies the path of an image/content
href="" - specifies the URL of the page a link goes to
rel="" - specifies the Path of a supportive document(CSS doc for example)
lang="" - specifies the lang of the page
style="" - add styles to an element
title="" - defines some extra info about an element

Headings : <h1> ... <h6> - search engines use headings to index the structure and content of the page, defines a level of importance of the elements
<br> : line break
<hr> : horizontal rule
<pre> : preformatted text, keeps spaces and line breaks

<figure>: can include image, multimedia..can include <figcaption></...> explaning the picture
<audio>: adds audio to the page
<video>: adds video to the page, can take controls attribute controls

<main>: contains the main content of the page
<header>: contains introdutory content about its nearest ancestor
<footer>: contain additional information about its nearest ancestor

<table>
<tr>: table rows
<td>: table columns
